no cloud · no sync · no telemetry

your money, on your machine.

kino is a local-first personal finance desktop app. track accounts, transactions, budgets, and net worth in a single .kino file you own.
store it on iCloud, Dropbox, or a thumb drive·nothing leaves your disk.

  • multi-account: checking, credit, investments, crypto
  • import ofx / qfx / csv with duplicate detection
  • budgets with rollover · weekly · monthly · annual
  • cash-flow sankey · 12 / 24-month net-worth history
  • cents as int64, never a floating-point dollar
  • one file on disk; back it up the way you like
